# rubik's cube

Rubik's Cube visualizer and solver.

## How to use
### Setup
#### Linux
```bash
sudo apt-get install libglfw3-dev
```
#### macOS
```bash
brew install glfw3
```
### Build
```bash
make
```
### Run
```bash
./bin/rubiks
```
### Clean
```bash
make clean
```
